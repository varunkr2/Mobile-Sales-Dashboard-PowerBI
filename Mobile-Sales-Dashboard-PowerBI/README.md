# рЯУК # Mobile Sales Dashboard | Power BI
## Project Overview
An interactive Power BI dashboard built to analyze mobile sales performance using Power
Query, Data Modeling, and DAX. The dashboard enables users to monitor key business
metrics, track Month-to-Date (MTD) performance, and compare sales with the Same
Period Last Year (SPLY) through interactive reports and visualizations.
